# рҹҸў AWS Organizations - Quick Setup Guide\n\nSetup AI Cloud Cost Detective to scan multiple AWS accounts in your AWS Organizations.\n\n---\n\n## What You Get\n\nвң… Scan **multiple AWS accounts** in your organization  \nвң… **Region selection** in the UI  \nвң… **Environment configuration** through the frontend  \nвң… **Cross-account access** using IAM roles  \nвң… **Local development** with Docker  \n\n---\n\n## 3-Step Setup\n\n### Step 1: Create Cross-Account Role in Child Accounts (5 min)\n\nIn **each AWS account** you want to scan:\n\n1. Go to **IAM Console** вҶ’ **Roles** вҶ’ **Create Role**\n2. Choose:\n   - **Trusted entity type**: AWS account\n   - **Account ID**: Your management account ID\n3. Add **Inline Policy**:\n\n```json\n{\n  \"Version\": \"2012-10-17\",\n  \"Statement\": [\n    {\n      \"Effect\": \"Allow\",\n      \"Action\": [\n        \"ec2:Describe*\",\n        \"rds:Describe*\",\n        \"s3:List*\",\n        \"lambda:List*\",\n        \"ce:GetCostAndUsage\"\n      ],\n      \"Resource\": \"*\"\n    }\n  ]\n}\n```\n\n4. Name it: `CostDetectiveRole`\n5. **Copy the role ARN** (looks like: `arn:aws:iam::123456789012:role/CostDetectiveRole`)\n\n### Step 2: Create aws_accounts.json (2 min)\n\nCreate `aws_accounts.json` in project root:\n\n```json\n{\n  \"accounts\": [\n    {\n      \"name\": \"Production\",\n      \"account_id\": \"123456789012\",\n      \"role_arn\": \"arn:aws:iam::123456789012:role/CostDetectiveRole\",\n      \"regions\": [\"us-east-1\", \"us-west-2\"]\n    },\n    {\n      \"name\": \"Development\",\n      \"account_id\": \"210987654321\",\n      \"role_arn\": \"arn:aws:iam::210987654321:role/CostDetectiveRole\",\n      \"regions\": [\"us-east-1\"]\n    }\n  ]\n}\n```\n\n### Step 3: Setup and Run (3 min)\n\n```bash\n# Create .env\ncat > .env << EOF\nAWS_ORGANIZATIONS_ENABLED=true\nAWS_MANAGEMENT_ACCOUNT_ID=your_management_account_id\nOPENAI_API_KEY=your_openai_key\nJWT_SECRET=your_secret\nEOF\n\n# Start services\ndocker-compose up -d\n\n# Access\nhttp://localhost:3000\n```\n\n---\n\n## UI Configuration\n\nWhen you open the app, you'll see:\n\n### 1.
